Para compilar o programa, utilize o comando 'make'. Antes de executar o programa, insira o arquivo de entrada no diretório pai do programa. Para executar, utilize o seguinte comando:
./tp3 <nome_arquivo_entrada> <metodo_casamento>
Substitua <metodo_casamento> pelo inteiro que representa o método de casamento de caracteres, onde: 1-Força bruta, 2-KMP, 3-BMH, 4-Shift-And.
Para limpar o executável e os arquivos objeto gerados, utilize o comando 'make clean'
